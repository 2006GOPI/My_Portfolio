# ‚ö° Quick Start Guide

Get your portfolio live in 15 minutes! This guide is for absolute beginners.

## üéØ What You'll Need

- A computer with internet
- A code editor (we recommend [VS Code](https://code.visualstudio.com/) - it's free!)
- Your content (bio, project info, photos)

## üìù 5-Minute Setup

### Step 1: Download & Open (2 minutes)

1. **Download this project**
   - Click the green "Code" button (if on GitHub)
   - Select "Download ZIP"
   - Extract the ZIP file to a folder

2. **Open in code editor**
   - Open VS Code
   - File ‚Üí Open Folder
   - Select your portfolio folder

### Step 2: Edit Your Info (3 minutes)

Open `index.html` and search for these (Ctrl+F or Cmd+F):

1. **Your Name** (appears 3 times)
   - Replace "Your Name" with your actual name

2. **Your Role** 
   - Line 68: Replace the hero description

3. **Email**
   - Search for "hello@yourname.com"
   - Replace with your email

4. **Social Links**
   - Search for "yourusername"
   - Replace with your actual usernames

### Step 3: Test Locally (1 minute)

**Easiest Method:**
1. In VS Code, right-click `index.html`
2. Click "Open with Live Server" (install extension if prompted)
3. Your portfolio opens in browser!

**Alternative:**
- Just double-click `index.html` to open in browser

## üöÄ Deploy to Internet (10 minutes)

### Method 1: Netlify (Recommended - Easiest!)

1. **Go to [Netlify](https://netlify.com)**
2. **Sign up** (free, use email or GitHub)
3. **Drag & drop** your portfolio folder onto Netlify
4. **Done!** You get a link like: `your-site-name.netlify.app`

**Bonus:** Netlify gives you:
- Free hosting forever
- Automatic HTTPS (secure)
- Working contact form
- Easy updates (just drag new files)

### Method 2: GitHub Pages (Good for learning Git)

1. **Create [GitHub](https://github.com) account** (free)
2. **Create new repository**
   - Click "+" ‚Üí "New repository"
   - Name it: `your-username.github.io`
   - Make it Public
3. **Upload your files**
   - Click "uploading an existing file"
   - Drag all your portfolio files
   - Click "Commit changes"
4. **Enable Pages**
   - Settings ‚Üí Pages
   - Source: main branch
   - Save
5. **Visit your site:** `https://your-username.github.io`

## üÜò Common Questions

### Q: How do I edit the code?
**A:** Use any text editor (VS Code recommended). HTML files are just text files!

### Q: I broke something. How do I fix it?
**A:** Re-download the original files and start over. It's okay!

### Q: Where do I put my photos?
**A:** In the `images/` folder. Then update the path in HTML.

### Q: The form doesn't work!
**A:** That's normal! Deploy to Netlify and it will work automatically.

### Q: Can I change colors?
**A:** Yes! See CUSTOMIZATION_GUIDE.md for color instructions.

### Q: How do I update after deploying?
**A:** 
- **Netlify:** Just drag updated files again
- **GitHub:** Upload changed files again

### Q: Do I need to know coding?
**A:** No! Just follow this guide to change text and images.

## üí° Pro Tips

1. **Start Simple**
   - Just change text and images first
   - Don't worry about code yet

2. **Save Often**
   - Press Ctrl+S (or Cmd+S) after changes
   - Refresh browser to see updates

3. **One Change at a Time**
   - Make one change
   - Test it
   - Then make next change

4. **Use Placeholder Photos**
   - The site already has placeholder images
   - Replace them when you have your own

5. **Deploy Early**
   - Don't wait for perfection
   - Get it online, then improve
